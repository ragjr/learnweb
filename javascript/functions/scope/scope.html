Introduction to JS Scope

Scope is the context that code is executed in.

        
            function doMath() {
                var x = 40;
                console.log(x);
            }

Executing doMath() returns 40 but x doesn't live in the global scope.

        
            var y = 99;

            function doMoreMath() {
                y = 100;
                console.log(y);
            }

Executing doMoreMath() returns y as 100 and changes y in the global scope to 100, because we didn't declare it as a new variable in the function.
